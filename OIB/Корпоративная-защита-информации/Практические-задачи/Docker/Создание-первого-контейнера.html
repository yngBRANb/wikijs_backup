Шаг 1: Установка Docker
Перед тем как начать, убедитесь, что Docker установлен на вашем компьютере. Вы можете скачать и установить Docker с официального сайта.

Шаг 2: Проверка установки Docker
После установки проверьте, что Docker работает, выполнив следующую команду в терминале:

docker --version

Вы должны увидеть версию установленного Docker.

Шаг 3: Запуск первого контейнера
Для создания и запуска первого контейнера используем образ hello-world, который предназначен для проверки установки Docker.

Шаг 4: Запуск контейнера
Выполните следующую команду:

docker run hello-world

Что происходит?

  • Docker проверит, есть ли у вас локально образ hello-world.
  • Если образ отсутствует, Docker автоматически загрузит его из Docker Hub.
  • После загрузки Docker создаст контейнер и выполнит его, выводя приветственное сообщение.

Шаг 5: Проверка запущенных контейнеров
Чтобы увидеть список запущенных контейнеров, выполните команду:

docker ps

Если вы хотите увидеть все контейнеры, включая остановленные, используйте:

docker ps -a

Шаг 6: Остановка и удаление контейнера
Если вы хотите остановить контейнер, выполните команду:

docker stop <идентификатор_контейнера>

Чтобы удалить контейнер, используйте:

docker rm <идентификатор_контейнера>

Заключение
Вы успешно создали и запустили свой первый контейнер в Docker! Теперь вы можете экспериментировать с другими образами и командами Docker.
