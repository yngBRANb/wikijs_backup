Вводная информация

Условно инструменты атаки можно разделить на несколько категорий:

  1. Сетевые сканеры
  2. Сканеры брешей в веб-скриптах
  3. Эксплойтинг
  4. Автомазация инъекций
  5. Дебаггеры (снифферы, локальные прокси и т.п.)

Каждая категория выполняет свое предназначение, очевидно что вы не будете пытаться порезать колбасу вилкой. 

Некоторые из указанных инструментов имеют "универсальный характер". Поэтому будут относиться к той категории в которой они имеют наибольший “результат” при применении. 

Сетевые сканеры

  1. Nmap - Основная задача — раскрыть доступные сетевые сервисы, установить их версии, определить ОС. Nmap («Network Mapper») бесплатный инструмент, имеет аналог Zenmap, который по факту является тем же самым только с графическим интерфейсом.Это не просто «умный» сканер, это серьезный расширяемый инструмент (из «необычных фишек» — наличие скрипта для проверки узла на наличие червя "Stuxnet" (упоминалось тут). Типовой пример использования:

    nmap -A -T4 localhost
  2. IP-Tools IP-Tools — эдакий набор из разных сетевых утилит, поставляется с GUI, «посвящена» windows юзерам.
    Сканер портов, общих ресурсов (расшаренные принтеры/папки), WhoIs/Finger/Lookup, telnet клиент и многое другое. Просто удобный, быстрый, функциональный инструмент.
