Создание нового репозитория с помощью CLI

git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin git@github.com:<username>/<repo_name>.git
git push -u origin main

push существующего репозитория из командной строки:

git remote add origin git@github.com:<username>/<repo_name>.git
git branch -M main
git push -u origin main
