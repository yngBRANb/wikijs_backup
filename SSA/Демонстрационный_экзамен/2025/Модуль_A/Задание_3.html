Создайте пользователя sshuser на серверах HQ-SRV и BR-SRV 

Для создания пользователя sshuser с паролем P@ssw0rd и идентификатором 1010 используйте следующие команды:

useradd sshuser -u 1010 
passwd sshuser

Чтобы он мог выполнять sudo без ввода пароля, необходимо отредактировать файл sudoers или создать новый в директории /etc/sudoers.d:

vim /etc/sudoers.d/sshuser

Введите в этот файл:

sshuser ALL=(ALL) NOPASSWD: ALL

После этого добавьте пользователя sshuser в группу суперпользователей - wheel:

usermod -aG wheel sshuser

Создайте пользователя net_admin на маршрутизаторах HQ-RTR и BR-RTR:

Создайте пользователя net_admin с помощью следующих команд:

vesr# config
vesr(config)# username net_admin
vesr(config-user)# password P@$$word
vesr(config-user)# privilege 15
vesr(config-user)# end
vesr# commit
vesr# confirm
