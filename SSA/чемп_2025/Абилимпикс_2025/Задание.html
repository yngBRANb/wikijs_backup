Подготовка оборудования 

  1. На R1 и R2
  •  Скачать последний стабильный образ OpenWrt с официального сайта. 
  • Перепрошить роутеры через веб-интерфейс (LuCI) или консоль (SSH).
  • Убедиться в успешной загрузке новой прошивки.

Обожмите кабель для соединения устройств, количество посчитайте по топологии 

Базовая настройка 

Назначить адреса согласно таблице: 

R1: 

  • DHCP (WAN-интерфейс) 
  • LAN: 192.168.120.1/24, 192.168.101.254/24 
  • Соединение с R2: 10.10.10.1/30 

R2: 

  • Соединение с R1: 10.10.10.2/30 
  • LAN: 192.168.110.1/24, 192.168.102.1/24 
  • ADMIN_PC: 192.168.101.10/24 

CLI: 

  • DHCP (получает адрес от R1) 

Устройства: 

  • cam1 (192.168.120.25) 
  • cam2 (192.168.110.25) 
  • print 13 (192.168.102.10) 

Необходимо настроить FQDN (полное доменное имя) для каждого устройства. 

Подключение к провайдеру 

Настройка DHCP на R1 

  • Настроить WAN-интерфейс для получения IP-адреса от провайдера через DHCP. 
  • Убедиться в наличии интернет-доступа через команду ping 8.8.8.8. 

Динамическая трансляция адресов (NAT) 

Настройка NAT на R1

  • Включить маскарадинг для LAN-сетей. 
  • Проверить доступ в интернет с клиентских устройств (например, CLI). 

DHCP-сервер для клиентов

Настройка DHCP на R1 

  • Настроить пул адресов для сети 192.168.101.0/24: 
  • Диапазон: 192.168.101.100–192.168.101.200. 
  • Исключить шлюз (192.168.101.254). 
  • Указать DNS-суффикс au.team и DNS-сервер (192.168.101.254). 
  • На клиентах (CLI) активировать получение IP через DHCP. 

Динамическая маршрутизация OSPF 

Настройка OSPF на R1 и R2 

  • Настроить OSPF-процесс с общими сетями: 
  • R1: 192.168.120.0/24, 192.168.101.0/24, 10.10.10.0/30. 
  • R2: 192.168.110.0/24, 192.168.102.0/24, 10.10.10.0/30. 

Сервер видеозаписи на ADMIN_PC

Установка и настройка 

  • Установить MySQL и создать базу данных для ZoneMinder. 
  • Создать пользователя zmuser с паролем P@ssw0rd2025. 
  • Настроить часовой пояс Asia/Yekaterinburg. 
  • Добавить камеры cam1 (192.168.120.25) и cam2 (192.168.110.25). 
  • Настроить зоны активации записи и хранение данных в /srv/video (7 дней). 
  • Сменить язык интерфейса на русский в настройках ZoneMinder.

 Сервер печати CUPS 

Настройка на ADMIN_PC 

  • Установить CUPS и настроить общий доступ к принтеру print (192.168.102.10). 
  • Запретите печать из интернета. 
  • Включить журналирование (максимум 300 заданий). 
  • Создать пользователя print с паролем P@ssw0rd. 
  • На клиенте CLI добавить сетевой принтер Сохранять логи печати в /srv/log_print.
