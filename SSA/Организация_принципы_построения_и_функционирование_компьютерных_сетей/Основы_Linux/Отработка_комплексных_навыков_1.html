Задания для практической работы:

  1. Необходимо создать директорию testdir в домашнем каталоге пользователя (/home).
  2. Одной командой создать в директории /home/testdir директории ..dir1/dir2/dir3.
  3. Удалить директорию /home/testdir/dir1/dir2 одной командой.
  4. С помощью команды cd переместитесь в любую директорию 3 уровня по абсолютному пути.
  5. С помощью команды cd перейти в материнскую директорию той директории в которой вы находитесь на данный момент.
  6. С помощью команды cd по относительному пути вернитесь в прошлую директорию.
  7. С помощью команды cd вернитесь в прошлую директорию не по относительному пути.
  8. C помощью команды cd вернитесь в домашний каталог вашего пользователя.


Контрольные вопросы:

  1. С помощью какой команды может быть получена информация о вашем текущем местонахождении в рамках файловой системы?
  2. Какой символ используется в начале пути файловой системы для указания абсолютного пути?
  3. Какая утилита предназначена для вывода списка содержимого директории?
  4. Какой параметр утилиты ls предназначен для вывода информации обо всех файлах, включая скрытые файлы?
  5. Какой ключ нужно передать команде mkdir, чтобы создать несколько вложенных каталогов одной командой?
  6. Напишите команду для создания структуры директорий /home/box/foo/bar/baz, если вы находитесь в домашней директории (/home/box).
