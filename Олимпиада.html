Kalolympics 2025 Winter Season

Kalolympics 2025 Winter Season — это уникальная олимпиада, посвященная сетевому и системному администрированию. Cоревнования будут состоять из трех модулей, которые продлятся 1 час 20 минут.

Первый модуль включает тестирование на знание теории, где участники смогут проверить свои знания в области сетевых протоколов, системного администрирования. Второй модуль сосредоточен на решении CTF (Capture The Flag) задач по информационной безопасности.

Третий модуль посвящен развертыванию игрового сервера, где участники смогут продемонстрировать свои способности в настройке и управлении серверной инфраструктурой.

Модуль 3

Разверните Minecraft-сервер на ubuntu.

Для подключения к удаленному серверу используйте команду:

ssh root@kalolympics.ru -p xxx

xxx - это выданный вам порт

В качестве пароля используйте P@ssw0rd


Требования:

  • Java должна быть версии openjdk-21-jre-headless
  • Используйте ядро: Paper 1.21.4 (его нужно самостоятельно скачать)
  • Сервер должен использовать стандартный порт (25565)
  • К серверу можно подключится с пиратской версии клиента minecraft (online-mode=false)

Для тестирования можете использовать веб-клиент. 
Подключаться к серверу для теста нужно по адресу kalolympics.ru:<тут_номер_вашего_api_порта>)
